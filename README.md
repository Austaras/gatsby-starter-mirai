# gatsby-starter-mirai

A Gatsby.js port of famous hexo theme [Next](https://github.com/theme-next/hexo-theme-next), with scheme Pisces

## Set up

1. Clone this repo
2. `yarn`
3. `yarn install`
4. rename config.example.yml to config.yml and modify it to your need
